Get started with Custom Fields in just a few steps.

## Requirements

- **PHP**: 8.2 or higher
- **Laravel**: 10.0 or higher (12.x fully supported)
- **Filament**: 3.0 or higher (4.x fully supported)

## Installation

Choose your installation method based on your license type:

<Tabs>
  <Tab title="Open Source (AGPL-3.0)">
    
    ### For open source projects

    Install directly from Packagist:

    ```bash
    composer require relaticle/custom-fields:^2.0
    ```

    Then run the installer:

    ```bash
    php artisan custom-fields:install
    ```

    <Warning>
    **Important**: By using this method, your entire application must be open source under AGPL-3.0. This includes SaaS applications.
    </Warning>

  </Tab>
  <Tab title="Commercial License">
    
    ### For commercial/private projects

    <Steps>
      <Step title="Purchase a license">
        [Choose a commercial license](/legal-acknowledgments/license#commercial-license-pricing) starting at $39/year
      </Step>
      
      <Step title="Add the private repository">
        Add our private Composer repository to your `composer.json`:
        
        ```json
        {
            "repositories": [
                {
                    "type": "composer",
                    "url": "https://satis.relaticle.com"
                }
            ]
        }
        ```
      </Step>
      
      <Step title="Install the package">
        ```bash
        composer require relaticle/custom-fields:^2.0
        ```
        
        When prompted for authentication:
        - **Username**: Your email (used for purchase)
        - **Password**: Your license key
      </Step>
      
      <Step title="Run the installer">
        ```bash
        php artisan custom-fields:install
        ```
      </Step>
    </Steps>

    <Note>
    Your license key will be saved in Composer's auth file for future updates.
    </Note>

  </Tab>
</Tabs>

## Optional Configuration

### Publish Configuration File

```bash
php artisan vendor:publish --tag="custom-fields-config"
```

### Publish Language Files

```bash
php artisan vendor:publish --tag="custom-fields-translations"
```

### Publish Views (for customization)

```bash
php artisan vendor:publish --tag="custom-fields-views"
```
