## What is Custom Fields?

Custom Fields is a powerful Laravel package that enables dynamic field management in Filament applications without requiring database migrations. Version 2 represents a complete architectural overhaul, introducing enterprise features like multi-tenancy, conditional visibility, and advanced import/export capabilities.

## What's New in Version 2

Version 2 is a major release with significant improvements over V1:

### Enhanced Architecture
- **Filament 4 Support** - Built from the ground up for Filament 4
- **Laravel Data Integration** - Type-safe DTOs using Spatie's Laravel Data
- **Entity Management System** - Automatic discovery and configuration of entities
- **Performance Optimizations** - Cached entity discovery and optimized queries

### Enterprise Features
- **Multi-Tenancy** - Full tenant isolation with automatic scoping
- **Conditional Visibility** - Show/hide fields based on complex logic
- **Import/Export** - CSV support with intelligent value conversion
- **Field Migrations** - Define fields programmatically via migrations

### New Field Types
- **Currency Field** - Dedicated currency input with formatting
- **Link Field** - URL validation and formatting
- **Toggle Buttons** - Radio-style toggle selections
- **Tags Input** - Tag entry with auto-completion
- **Markdown Editor** - Rich markdown editing support

### Developer Experience
- **Type Safety** - Full type safety with DTOs and enums
- **Service Architecture** - Modular service providers
- **Extensibility** - Clean interfaces for customization
- **Modern PHP** - Requires PHP 8.3 for latest language features

## Use Cases

Custom Fields V2 is ideal for applications that need:

- **CRM Systems** - Add client-specific fields without code changes
- **E-commerce** - Dynamic product attributes per category
- **Multi-tenant SaaS** - Tenant-specific customizations
- **Enterprise Applications** - Department-specific data fields
- **Workflow Systems** - Stage-specific field requirements

## System Requirements

- PHP 8.3 or higher
- Laravel 10.0 or higher
- Filament 4.0 or higher
- MySQL 8.0+ / PostgreSQL 13+ / SQLite 3.35+
